# modern-cpp-tutorial
Source code for slides on C++ a tutorial/overview of Modern C++

# BUILD

## CentOS 8
    `make STD=c++14 -j4`

## CentOS 7
    `make STD=c++11 -j4`

You can also try other settings for STD on each platform.
